# My Cypress Automation

This project uses Cypress to automate testing of a login feature and a user update feature. Cypress is a JavaScript-based end-to-end testing framework that allows developers to write tests and execute them in a real browser environment.

## Installation

```bash
npm install
```
>NOTE:
>
>You need to have Node.js installed on your machine before running the installation.

## Running Tests

To open the Cypress Test Runner UI, run:
```bash
npx cypress open
````
To run the tests directly from the command line, run:
```bash
npx cypress run
```
## Features

Automated login test

Automated user update test
